# Editorial Layout

A fresh approach to magazine design that balances typography, imagery, and white space.

## Project Details

Created a cohesive visual language for a quarterly art publication, emphasizing readability and visual impact.

### Design Elements

- Grid system development
- Typography hierarchy
- Image curation
- Layout experimentation

The design creates a rhythmic reading experience that complements the editorial content.
